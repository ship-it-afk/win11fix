# Win11Fix 🎨

A fully open-source Graphical User Interface (GUI) web tool that guide you to fix your Windows 11 computer. 

## 🚀 Features
* **Natural Language Guidance:** Clear, easy-to-follow instructions—no advanced technical knowledge required.
* **One-Click Solutions:** Direct links to Windows settings and built-in troubleshooters.
* **System Optimization:** Helpful tips to improve performance and remove bloatware.
* **Portable & Web-Based:** Run the guide directly in your browser without installing heavy software.

## 🛠 Installation
If you want to run Win11Fix locally:
1. Clone the repo: `git clone https://github.com/ship-it-afk/win11fix.git`
2. Open `index.html` in your browser (or run `npm install` if you have dependencies).

## 📖 How to Use
1. Launch the Tool: Open the hosted link (https://ship-it-afk.github.io/AWC) or your local index.html.
2. Identify the Issue: Identify your Windows 11 machine issue (eg. BSOD, Easily Crash, etc). 
3. Type issue in natural language. 
4. Follow the Steps: Use the interactive guide to apply fixes to your Windows 11 system.

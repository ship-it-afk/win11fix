# Security Policy

## Supported Versions

Below are the version that currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| 0.9.x   | :x:                |
| 0.5.x   | :x:                |
| < 0.1   | :x:                |

## Reporting a Vulnerability

Please report security vulnerabilities by opening a private Draft Security Advisory on GitHub or by contacting the maintainer via asastechnologies5@gmail.com.

Please include:

• A description of the vulnerability.

• Steps to reproduce the issue.

• The potential impact if exploited.

## What to Expect

Response Time: I will acknowledge your report within 48–72 hours.

Updates: I will provide progress updates every few days while a fix is being developed.

Disclosure: Once a fix is released, I will publicly disclose the vulnerability to protect other users, giving you full credit for the discovery (if desired).

## 🛡️ A Note on System Safety

Because Win11Fix and AndroidFix provides guides for system-level changes, please remember:

This tool should only be downloaded from the official repository: https://github.com/ship-it-afk/win11fix.

I will never ask for your Windows / Android password or remote access to your computer / phone via this project.
